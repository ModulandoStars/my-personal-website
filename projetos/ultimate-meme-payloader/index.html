Ultimate Meme Payloader, é um software cujo seu objetivo é fazer popup a cada tempo determinado de um SFX, meme, música aleatoriamente.

"Por que essa merda não sai?" - c0de.wtf

Preparado pra ser modular, deve aguentar a máxima quantidade de popups possível. Certamente é um app útil para assustar seus amigos

Github Repo

A História do desenvolvimento.

O projeto teve seu ínicio em meados de outubro de 2024, vendo a existência de vídeos de "1 hour of silence broken by [random meme]", eu pensei na ideia de criar um app que faz isso, só que a diferença sendo que é mais flexível que um vídeo do YouTube. Sendo assim o início de um simples app. E isso demorou mais que eu queria aceitar :)
